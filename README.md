# 🎵 YTMusic Importer

Импортируйте любой список треков в плейлист **YouTube Music** прямо из терминала.

⚠️ Скрипт использует **неофициальный API** [ytmusicapi](https://ytmusicapi.readthedocs.io/en/stable/). Данные пользователей **никуда не передаются и не сохраняются** скриптом — авторизационный токен хранится только в локальном файле `browser.json`.

## 🔑 Шаг 1 — Авторизация

Авторизация происходит через cookies браузера. Нужно один скопировать заголовки запроса из DevTools.

### 1.1 Открыть DevTools и поставить лайк

1. Перейдите на [https://music.youtube.com](https://music.youtube.com) и убедитесь, что вы **вошли в аккаунт**.
2. Откройте DevTools:
   - `F12` или `Ctrl+Shift+I`
3. Перейдите на вкладку **Network**.
4. Поставьте **лайк** любому треку — в Network появится POST-запрос.

### 1.2 Скопировать заголовки запроса

#### Firefox (рекомендуется)

Найдите запрос со статусом **200** и методом **POST** к домену `music.youtube.com`.  
ПКМ по запросу → **Copy → Copy Request Headers**.

#### Chrome / Edge

Найдите запрос с именем `browse?...` и статусом **200**.  
Нажмите на запрос → вкладка **Headers** → прокрутите до раздела **Request Headers** → скопируйте всё начиная с `accept: */*`.

> На новых версиях Chrome: ПКМ по имени запроса → **"Copy as fetch (Node.js)"**

### 1.3 Создать browser.json

Выполните в терминале:

```bash
ytmusicapi browser
```

Скрипт запросит вставку скопированных заголовков.

#### Windows

Просто вставьте: `Ctrl+V` или ПКМ в окне терминала. Нажмите `Ctrl + D`.

#### macOS

Из-за ограничения терминала (максимум 1024 символа за раз) **нельзя** вставить заголовки обычным `Cmd+V`.  
Используйте утилиту `pbpaste`, которая читает буфер обмена:

```bash
pbpaste | ytmusicapi browser
```

После успешного выполнения в папке появится файл **`browser.json`** — он содержит ваши учётные данные.

## 📄 Шаг 2 — Подготовить файл с треками

Создайте текстовый файл (например, `tracks.txt`). Каждая строка — один трек:

```
The Beatles - Hey Jude
Radiohead - Creep
Pink Floyd - Comfortably Numb
Daft Punk - Get Lucky
```

---

## 🚀 Шаг 3 — Запустить импорт

```bash
python main.py
```

В интерактивной консоли введите:

```
/start
```

Скрипт последовательно спросит:

1. **Имя файла с треками** — введите имя вашего файла (по умолчанию `tracks.txt`)
2. **Название плейлиста** — как назвать новый плейлист в YouTube Music

После этого начнётся автоматический поиск каждого трека и добавление в плейлист. Дождитесь завершения — в конце вы получите ссылку на готовый плейлист.

## 💾 Сохранение прогресса

Если импорт был прерван (`Ctrl+C` или ошибка сети), прогресс автоматически сохраняется в `checkpoint.json`.  
При следующем запуске `/start` скрипт предложит продолжить с места остановки.

## 📟 Доступные команды

| Команда  | Описание                           |
| -------- | ---------------------------------- |
| `/start` | Запустить / возобновить импорт     |
| `/help`  | Показать справку прямо в терминале |
| `/exit`  | Выйти из скрипта                   |
